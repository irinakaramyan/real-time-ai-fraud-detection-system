Real-Time AI Fraud Detection System

Overview
This project is a real-time fraud detection system for financial transactions.

The system processes transactions, calculates risk using rules and AI, and classifies them as:
- Approved
- Review
- Blocked

Technologies
- FastAPI
- PostgreSQL
- Redis
- Scikit-learn
