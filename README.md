AI Fraud Detection System
A web-based system that detects suspicious financial transactions using rule-based logic and AI techniques.

 Overview
This project aims to identify potentially fraudulent transactions by analyzing user behavior and transaction patterns.

 Features
- Transaction monitoring  
- Risk scoring  
- Suspicious activity detection  

Tech Stack
- Python  
- HTML, CSS, JavaScript  
- SQL  

Goal
To build a system that helps detect fraud in financial transactions.
